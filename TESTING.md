## 1. Как запустить приложение

### Вариант A: В Telegram (правильный способ)
1. Открыть Telegram на телефоне (Android или iOS)
2. Найти бот @KitaiskuiBot
3. Нажать `/start`
4. Нажать кнопку **«Начать заниматься»** внизу экрана
5. Откроется Mini App

### Вариант B: В браузере (для быстрого теста)
Открыть: https://repetitor-kitaiskui.vercel.app

**Отличия от Telegram-версии в браузере:**
- Вместо синей MainButton Telegram — появляется серая HTML-кнопка снизу
- Кнопка «Назад» — в браузере видна как синяя кнопка `← Назад` в правом верхнем углу
- HapticFeedback (вибрация) не работает — только в Telegram
- Голос работает только при наличии китайских TTS-голосов в системе

### Вариант C: Локально из папки (только для разработчика)
Открыть файл: `tg-app/index.html` в браузере.

**Важно:** При открытии через `file://` протокол `fetch()` не работает.
Приложение автоматически переключается на встроенные данные (`loadEmbeddedData()`).
Весь функционал работает — данных достаточно для тестирования.

## 2. Сброс прогресса перед тестом

Приложение сохраняет состояние в `localStorage` под ключом `repetitor_progress`.
При повторном открытии — сразу показывает дашборд (минуя онбординг).

**Чтобы начать с нуля:**

В браузере — открыть консоль (F12) и выполнить:
```js
localStorage.clear(); location.reload();
```

В Telegram — сброс через консоль невозможен. Данные также хранятся в `Telegram CloudStorage`.

## 3. Архитектура (что где лежит)

```
tg-app/
├── index.html        — все 17+ экранов как div.screen
├── app.js            — объект App, вся логика
├── style.css         — стили, Telegram CSS-переменные
├── api/
│   └── tts.js        — Vercel serverless: прокси Google TTS
└── data/
    ├── hsk1.json     — 150 слов HSK 1
    └── lessons.json  — 10 уроков, 4 категории
```

**Ключевые переменные состояния** (`App.state.user`):
| Поле | Тип | Описание |
|------|-----|----------|
| `goal` | string | Цель: study / work / travel / hobby |
| `level` | string | Уровень: zero / beginner / intermediate |
| `dailyGoal` | number | Минут в день: 5 / 10 / 15 / 20 |
| `onboardingDone` | boolean | Прошёл ли онбординг |
| `trialStartDate` | ISO string | Дата первого запуска (для 3-дневного триала) |
| `subscription` | string | free / basic / full / yearly |
| `xp` | number | Набранные очки опыта |
| `streak` | number | Дней подряд |
| `completedLessons` | array | ID пройденных уроков |
| `wordsLearned` | array | ID выученных слов |

## 4. Тест-кейсы

### TC-01: Онбординг (первый запуск)

**Предусловие:** Сброшен прогресс (`localStorage.clear()`)
**Среда:** Браузер или Telegram

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть приложение | Экран «Приветствие»: дракон 🐉, заголовок «Я твой репетитор по изучению китайского языка», кнопка «Начать обучение» |
| 2 | Нажать «Начать обучение» | Переход на экран «Зачем тебе китайский?» |
| 3 | Выбрать любую цель (например «Учёба / HSK») | Карточка подсвечивается, через 300мс переход на «Какой у тебя уровень?» |
| 4 | Выбрать уровень «Нулевой» | Переход на «Сколько минут в день?» |
| 5 | Выбрать «10 минут» | Переход на мини-урок |
| 6 | Мини-урок: ответить на 3 вопроса (你好, 谢谢, 再见) | После каждого ответа — подсветка зелёным/красным, анимация «+10 XP» при правильном |
| 7 | Завершить мини-урок | Конфетти 🎉, переход на дашборд |

**Что проверить в localStorage после:**
```js
JSON.parse(localStorage.getItem('repetitor_progress'))
// onboardingDone: true
// goal: "study"
// level: "zero"
// dailyGoal: 10
// trialStartDate: "<ISO дата>"
```

### TC-04: Триал — блокировка разделов

**Логика в коде** (`app.js`, метод `isTrialActive()`):
```js
// Если subscription !== 'free' — полный доступ
// Если с trialStartDate прошло < 3 дней — доступ открыт
// Если >= 3 дней — заблокированы все разделы кроме 'basics'
```

**Тест A: Триал активен (< 3 дней)**
| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть дашборд | Разделы «Разговорный», «Грамматика», «Деловой» — без замков, с зелёным бейджем «X дн. осталось» |
| 2 | Нажать «Разговорный» | Открывается список уроков |

**Тест B: Триал истёк (симуляция)**

В консоли браузера:
```js
// Устанавливаем дату 5 дней назад
const old = JSON.parse(localStorage.getItem('repetitor_progress'));
old.trialStartDate = new Date(Date.now() - 5*24*60*60*1000).toISOString();
localStorage.setItem('repetitor_progress', JSON.stringify(old));
location.reload();
```

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть дашборд | Разделы «Разговорный», «Грамматика», «Деловой» — с замком 🔒 |
| 2 | Нажать заблокированный раздел | В Telegram: всплывает popup «Пробный период завершён. Оформите подписку...» / В браузере: alert с тем же текстом |
| 3 | Раздел «Основы» | Работает без ограничений |

### TC-06: Голос (TTS)

**Как работает:** `App.speak(text)` → запрос на `https://repetitor-kitaiskui.vercel.app/api/tts?text=你好` → Vercel прокси → Google Translate TTS → MP3

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Telegram Android: нажать 🔊 у иероглифа | Слышен голос — женский китайский |
| 2 | Браузер Chrome/Яндекс: нажать 🔊 | Слышен голос |
| 3 | Настройки: выключить «Звук произношения» → нажать 🔊 | Звук НЕ воспроизводится |
| 4 | Включить звук обратно | Звук снова работает |

**Прямая проверка API:**
```
https://repetitor-kitaiskui.vercel.app/api/tts?text=你好
```
Должен вернуть аудиофайл (Content-Type: audio/mpeg).

## 5. Известные ограничения (не баги)

| Ограничение | Описание |
|-------------|----------|
| Голос не работает в Telegram Desktop (Windows) | Web Speech API не имеет китайских голосов без языкового пакета Windows |
| Голос работает в Яндекс.Браузере | Браузер имеет встроенные TTS-голоса |
| Голос работает в Telegram на Android | Используется Vercel TTS-прокси |
| Оплата — заглушка | `showPayment()` показывает только уведомление «Скоро» |
| AI-чат — скриптовый | Реальный Claude API не подключён |
| Подразделы ступеней — пустые | Контент будет добавлен позже |
| Триал сбрасывается при очистке localStorage | `trialStartDate` хранится только локально |
