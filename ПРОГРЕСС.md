# Статус проекта — Репетитор Китайский
Последнее обновление: 2026-04-06

## ✅ Готово

### Telegram Mini App
- Работает на https://repetitor-kitaiskui.vercel.app
- Онбординг, дашборд, уроки, упражнения, AI-чат (заглушка)
- 3 ступени углублённого изучения (подразделы пустые — ждут контент)
- Голосовое произношение работает в Telegram на Android

### Telegram Бот @KitaiskuiBot
- Запущен через PM2 (работает постоянно)
- Ученик: /start /buy /paid /progress /homework /help
- Учитель: /students /homework /student /add_lesson
- Админ (вы): /unlock /lock /assign /add_teacher /stats /payments
- Оплата: ученик /paid → вам кнопка [✅ Открыть доступ]
- База данных: bot/data.json (создаётся автоматически)

### Инфраструктура
- GitHub: https://github.com/idiirinasmr/repetitor-kitaiskui
- Vercel: авто-деплой при каждом git push
- Node.js v24.14.1 + PM2 установлены

## ⏳ Следующий шаг — веб-платформа с уроками

**Ситуация:** видеоуроки лежат на Google Диске, нужно выбрать где хранить и как показывать.

**Варианты хранения видео:**
| Вариант | Цена | Надёжность в РФ |
|---|---|---|
| YouTube (закрытые видео) | Бесплатно | Иногда блокируют |
| VK Видео | Бесплатно | Надёжно |
| Google Диск | Бесплатно (15 ГБ) | Медленный плеер |
| Kinescope / Boomstream | от 1500 ₽/мес | Отлично |

**После выбора:** создать страницы уроков на Vercel, доступ по ссылке после оплаты.
