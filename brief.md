# Brief: Репетитор по китайскому — Telegram Mini App

> Обновлено: март 2026
> Статус: MVP опубликован, идёт тестирование на клиентах

---

## Что это такое

Telegram Mini App — приложение внутри Telegram, которое открывается кнопкой у бота @KitaiskuiBot.
Выглядит как нативное мобильное приложение, не как сайт.
Цель: продавать обучение китайскому языку через подписку или разовую покупку курсов.

---

## Общие требования к дизайну

- Дизайн нативный Telegram (CSS-переменные `--tg-theme-*`)
- Крупные кнопки минимум 48px (touch target)
- Минимум текста, максимум визуала
- Русский язык интерфейса
- Иероглифы 28-40px, пиньинь 14px сверху
- Цветовая кодировка тонов: 1-красный, 2-оранжевый, 3-зелёный, 4-синий
- Загрузка < 1 сек
- Бандл < 100KB (vanilla JS, без фреймворков)
- Шрифты: Inter (UI) + Noto Sans SC (иероглифы)
- BackButton работает на всех страницах
- Звук ТОЛЬКО по нажатию 🔊 (не автоплей)

## Система учитель ↔ ученик (планируется)

### Как ученик сдаёт ДЗ:
1. Открывает урок в боте
2. Видит кнопку «Сдать домашнее задание»
3. Отправляет: текст / голосовое / фото
4. Бот принимает и пересылает учителю
5. Ученик видит статус: «Отправлено, ждёт проверки»

### Как учитель видит ДЗ:
1. Приходит сообщение в личный Telegram:
   ```
   📩 ДЗ от Анны (@anna)
   Урок: Ступень 1 — Фонетика
   [голосовое 0:23]
   ```
2. Учитель слушает / читает
3. Пишет ответ боту → бот пересылает ученику

### Команды учителя в боте:
```
/students  → список всех учеников + прогресс
/homework  → непроверенные ДЗ
/unlock @username ступень2 → открыть ступень после оплаты
```

### Где хранится прогресс учеников:
- Google Таблица (бесплатно)
- Бот автоматически записывает: кто, когда, что прошёл, статус ДЗ
