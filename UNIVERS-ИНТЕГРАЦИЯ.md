## Решение принято

**Платформа Univers под этот проект:** ниша **Репетиторство**

Почему репетиторство, а не онлайн-школа:
- Живые учителя проверяют домашние задания
- Каждому ученику присваивается учитель лично
- Есть индивидуальная обратная связь

## Что добавит Univers Platform

| Что | Зачем |
|---|---|
| Расписание занятий | Онлайн-запись к преподавателю |
| CRM-воронка | Новый → Пробный → Оплатил → Учится |
| Оплата ЮKassa | Автоматически, без ручного перевода |
| AI-агент на сайте | Замена скриптового чата на Claude |
| Кабинет ученика | Прогресс, задания, история |

## Следующие шаги (по порядку)

1. **Деплой Univers Platform на Vercel**
   - Репо: `idiirinasmr/univers-platform`, папка `app/`

2. **Supabase** — база данных для Univers

3. **Адаптировать Univers под нишу "Репетиторство по китайскому"**
   - Название, цвета, контент под проект

4. **Подключить Claude API**
   - Заменить скриптовый чат в боте на настоящий AI

5. **Подключить автооплату ЮKassa**
   - Убрать ручное подтверждение платежей

6. **Интегрировать бот с Univers CRM**
   - Заявки из бота → в CRM платформы
